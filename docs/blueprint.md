# **App Name**: Virtual Atheneum

## Core Features:

- Library View: Display a visually rich, dark-themed virtual library environment.
- Bookshelf Categorization: Allow users to browse their collection by pre-defined categories.
- Book Display: Showcase books as covers on shelves, including titles.
- Book Details: Open a side panel with detailed book information.
- External Platform Link: Provide a direct link to open the book on an external reading platform (KOBO, Readmoo).
- Book Data Import: Enable importing book data, prioritizing direct API connection to KOBO/Readmoo if available; otherwise, implement manual JSON import as a fallback.

## Style Guidelines:

- Primary color: Dark olive (#2D1B0D) to enhance the classic library theme.
- Background color: Very dark olive (#D4AF37) to enhance the dark theme.
- Accent color: Gold (#F3E5AB) for highlights and interactive elements to give a sense of sophistication.
- Headline font: 'Playfair', a serif, for headers and titles to create an elegant and fashionable impression; body font: 'PT Sans', a sans-serif, for descriptions and details to ensure readability.
- Smooth transitions with fade-in/fade-out effects and gentle movements when switching shelves or displaying book details.
- A layout that mimics the look and feel of an old European library, using wood texture and other design details to enhance the atmosphere.
